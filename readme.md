### Conways Game of Life CE

A Implemenation of Conways Game of Life for the CE in C.
Complete with a in-game editor and saving.

(also the in-game menu does not scrool that fast anymore.)

## Controls

### Menu Navigation
- **Up / Down** — Change the selected menu option
- **Enter** — Confirm selection

### Mode Switching
- **Enter** — Toggle between *Editing Mode* and *Running Mode*
  - The simulation only advances while in **Running Mode**

### Editing Mode
- **2nd** — Toggle the cell under the cursor (alive ↔ dead)
- **Arrow Keys** — Move the cursor around the grid
- **Del** — Open the in‑game menu *(quirk: only works in Editing Mode for now)*

### Global Controls
- **Clear** — Exit the program at any time
